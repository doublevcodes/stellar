# Stellar

This project will provide users with an interactive guide of the solar system.

## What Stellar uses

- Stellar uses [SvelteKit](https://kit.svelte.dev/) for the frontend. It is a
  framework that provides a simple way to build interactive web apps.
- Stellar also uses [TailwindCSS](https://tailwindcss.com/) for the styling.
  TailwindCSS is a CSS-in-JS library that provides a simple way to build CSS styles
